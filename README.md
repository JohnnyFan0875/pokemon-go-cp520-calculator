# PokemonGo CP Calculator

## About
This program collects stats of all staged evolutionary Pokémon in Pokémon GO by a given CP (Combat Power) value. It helps trainers analyze and track the stats of Pokémon evolutions relative to a specified CP threshold.

## Features
- Calculates stats for Pokémon at different evolution stages based on a target CP value.
- Supports batch processing of Pokémon stats.
- Outputs detailed stats useful for gameplay optimization and collection tracking.
